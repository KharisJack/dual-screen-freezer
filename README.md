# dual-screen-freezer
DSF Tool - 双屏等比定屏工具 - DualScreenFreezer
# 项目介绍
DSF 是一款允许你在使用电脑连接显示器时，当存在 2 个显示器时，你可以切换至【冻结模式】，使第二块屏幕的画面被冻结，并且不影响主设备操作。也就是说，此时副屏画面被冻住，而你依然可以任意在电脑上操作而不受影响。
# 使用场景
## 教学场景
- 中国大陆的学校，教学通常使用一体机设备。搭配常为墙上悬挂一体机，并接线连至讲台显示器便于教师学生操作电脑。很多时候学生需要抄写内容，教师需要调换课件，但是不希望影响到其他同学的观看，就可以通过这个工具把大屏幕的画面冻住。
- `ClassIn 品牌`的教学一体机暂不支持像 `SEEWO 一体机` 通过内嵌安卓系统的`涂鸦功能`冻结屏幕，就可以使用本工具。且本工具使用便利，完成智能向导配置好环境后，需要时仅需按下 `F1` 键即可冻结，`F2` 键即可解冻。
# 功能名词
## 冻结模式
冻结模式了是指 2 块屏幕内容相互独立，主屏幕 `主显示器`可以任意操作，同时副屏 `扩展屏` 画面保持不变。
## 解冻模式
解冻模式了是指 2 块屏幕内容保持一致，主屏幕 `主显示器`与副屏 `扩展屏` 画面保持一致，恢复原状。
# 基本原理
- 识别到用户按下快捷键 `F1` 时，程序执行截图操作，同时切换至扩展 `extend` 模式，并把截图在扩展副屏上以中心为锚点放置，此时 2 块屏幕的画面相对独立，互不干扰。
- 识别到用户按下快捷键 `F2` 时，程序切换至复制 `clone` 模式，同时执行销毁截图操作，此时恢复 2 块屏幕画面同步。
# 开发环境
- 本程序基于 AutoHotkey V2.0 构建编译，由作者设计基础运行逻辑，由 Google Gemini 完成代码编写。
