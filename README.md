## 📝 项目介绍

**DSF (DualScreenFreezer)** 是一款允许你在使用电脑连接外部显示器（且系统存在 2 个显示器）时，一键切换至【冻结模式】的生产力工具。

它可以**使第二块屏幕（如教室大屏、投影仪）的画面瞬间被冻结，同时完全不影响主设备（如讲台电脑）的操作**。此时副屏画面被死死冻住，而你依然可以任意在主电脑上查阅资料、修改分数或处理隐私信息，台下观众、学生毫无察觉。

---

## 🏫 典型使用场景：多媒体教学

- **中国大陆的中小学/高校教室**：教学通常使用一体机设备，配置常为墙上悬挂的交互式大屏一体机，并通过线缆连接至讲台的显示器，便于教师面向学生操作电脑。很多时候学生需要看着大屏抄写内容、订正答案，而教师需要调换下一个课件、登录微信接收文件、甚至批改其他同学的分数。此时，通过 DSF 工具把大屏幕画面冻住，即可完美解决隐私外泄和教学中断的痛点。
- **ClassIn 一体机的硬核平替**：诸如 `SEEWO (希沃) 一体机` 通常可以通过内嵌安卓系统的“涂鸦/冻结”功能来锁住屏幕，但 `ClassIn 品牌` 等许多教学一体机或常规会议室投影仪**暂不支持硬件定屏**。DSF 完美填补了这一空白。它不依赖任何特定硬件品牌，完成智能向导配置后，需要时仅需按下 `F1` 键即可定屏，`F2` 键即可解冻，极度便利。

## 📖 功能名词解释

- **❄️ 冻结模式**：指 2 块屏幕内容相互独立。主屏幕（主显示器）可以任意点击操作、切换窗口；同时副屏（扩展大屏）画面保持在按下 F1 那一刻的状态不变。
- **📺 解冻模式**：指 2 块屏幕内容恢复原状、保持一致。主屏幕（主显示器）与副屏（扩展屏）的画面实时同步镜像。

## 🛠️ 核心功能与进阶特性

除了核心的定屏功能，针对日常教学中经常遇到的“鼠标找不到”、“软件在大屏打开拉不回来”等突发状况，DSF 内置了多项硬核守护功能：

1. **等比无损定屏 (F1) / 镜像解冻 (F2)**：一键瞬时响应，资源秒级回收。
2. **副屏应用引力疏散 (F3)**：全网清洗系统窗口句柄，一键将所有不小心启动或被甩到“副屏黑屏域”内的应用窗口强制拉回主屏幕，再也不用盲操把软件往回拽。
3. **鼠标左右键联动归位**：在任意时刻**同时按下鼠标左键和右键**，光标将无感瞬间瞬移回主屏幕正中央，解决在大屏幕上找不到鼠标指针的尴尬。
4. **自适应双模故障排错 (托盘菜单)**：
   - **定屏状态下点击**：自动激活【高兼容物理像素降级装甲】，彻底绕过 Windows 所有的 DPI 虚拟化欺骗，原地一键重构画布，彻底根除小部分魔改系统下高 DPI 缩放引起的“仅冻结左上角”顽疾。
   - **常规状态下点击**：执行系统级环境自清洗，强力回拨镜像模式。
5. **内核级多实例硬拦截**：利用系统全局唯一 Mutex 信号量，彻底封堵因重复双击、跨路径启动引发的双实例冲突导致画面错位的问题。

## 📐 基本原理

- **按下 `F1` 时**：程序执行截屏操作，同时将系统切换至扩展（`extend`）模式，并把截图在扩展副屏上以中心为锚点平铺放大放置。此时 2 块屏幕的画面相对独立，互不干扰。
- **按下 `F2` 时**：程序将系统切换回复制（`clone`）模式，同时执行销毁截图和 GDI 内存释放操作，此时恢复 2 块屏幕的画面实时同步。

## 📥 安装与使用指南

本仓库提供的是**已经编译完成、独立绿色的 `DualScreenFreezer.exe` 可执行文件**。

1. **前往 [Releases] 页面**下载最新版本的 `DualScreenFreezer.exe`。
2. 本程序为**绿色便携版**，无需安装，不写注册表，直接双击即可运行。
3. **智能启动向导**：首次运行会弹出全动态 DPI 响应式聚焦向导面板，点击界面中的 **“⚙️ 自动检查并调整”**，程序会自动帮你检测并优化当前的双屏配置。
4. 在向导末页勾选“开机自动启动”，程序会在完成后自动将自身复制到系统启动目录，以后开机即用。

### ⌨️ 快捷键速查

| 快捷键 | 功能描述 | 适用场景 |
| :--- | :--- | :--- |
| **`F1`** | **进入全屏定屏冻结状态** | 准备切回主屏操作隐私内容或查阅资料时 |
| **`F2`** | **强制解除定屏，恢复双屏同步** | 随时可按，安全退出定屏状态 |
| **`F3`** | **大清洗：强制移回副屏的所有窗口** | 软件在大屏打开、PPT卡在后台点不到时 |
| **`F4`** | **重置脚本运行数据与自启动** | 更换教室、设备，或者需要重新进入向导时 |
| **`左键 + 右键`** | **鼠标光标无感瞬移回主屏中心** | 大屏幕太大，盲操找不到光标时 |
